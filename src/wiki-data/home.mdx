# Welcome to Warex's Wiki!
This is a place where anyone can modify, create or remove documentation, though actions like these are monitored.
This is a place where your documentation could get **promoted** to official documentation, so if you have something to share, don't hesitate to share it here!

# How to contribute?
To contribute to the wiki, you can simply click on the "Edit" button! Our wiki is built on top of Next.js, so you can edit the content as you would edit a normal markdown file. Our backend automatically converts the markdown content to HTML and saves it in our database, so you don't have to worry about that part. You can also add metadata to your documentation, such as the title, description, and tags, which will help other users find your documentation more easily.
